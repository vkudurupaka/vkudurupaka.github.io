Q6. What is QoS in Kubernetes?
QoS (Quality of Service) defines pod eviction priority during resource pressure. Classes are:
  • Guaranteed
  • Burstable
  • BestEffort
Q7. Which QoS class is evicted last?
Guaranteed pods are evicted last during resource starvation.
Q8. How do you configure requests and limits to make a Pod evicted last?
Set CPU and Memory requests equal to limits. resources: requests: cpu: "500m" memory: "512Mi" limits: cpu: "500m" memory: "512Mi"
Q9. What is a Liveness Probe?
A liveness probe checks if a container is running properly. If it fails, Kubernetes restarts the container.
Q10. What is a Readiness Probe?
A readiness probe checks if a container is ready to receive traffic. If it fails, traffic stops but the container is not restarted.
